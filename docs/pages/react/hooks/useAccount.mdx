# useAccount

This is a hook used to get the data associated with an account or the current connection status for that account. It allows for the collection of essential account information.

```ts
import { useAccount } from '@casperdash/usewallet';
```

## Usage

```tsx
import { useAccount } from '@casperdash/usewallet';

function App() {
  const { publicKey } = useAccount();

  return (
    <div>
      Public key: {publicKey}
    </div>
  )
}
```

## Return Value

```ts
{
  publicKey: string,
  status: string
}
```

## Configuration


### onConnect (optional)

Function to invoke when account connects.

```ts
import { useAccount } from '@casperdash/usewallet';

function App() {
  const { publicKey } = useAccount({
    onConnect(status) {
      console.log(status);
    },
  })
}
```

### onDisconnect (optional)

Function to invoke when account disconnects.

```ts
import { useConnect } from '@casperdash/usewallet';

function App() {
  const { connect } = useConnect({
    onDisconnect(status) {
      console.log(status);
    },
  })
}
```
