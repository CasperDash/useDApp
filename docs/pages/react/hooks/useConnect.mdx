# useConnect

Easy-to-use hook for quickly connecting to an account with compatible connectors. It provides a convenient way to quickly and securely sync accounts with a variety of services and applications.

```ts
import { useConnect } from '@useWallet/react';
```

## Usage

```tsx
import { CasperDashConnector } from '@casperdash/useWallet/react';
import { useConnect } from '@casperdash/useWallet/react';

function WalletProfile() {
  const { connect: connectWithCasperDash } = useConnect({
    connector: new CasperDashConnector(),
  });

  return (
    <div>
      <button onClick={() => connectWithCasperDash()}>Connect with CasperDash</<button>
    </div>
  )
}
```

## Return Value

```ts
{
  connect: function
}
```

## Configuration

### connector (Required)

Connector to use for connecting wallet

```tsx
function App() {
  const { connect } = useConnect({
    connector: new CasperDashConnector(),
  })
}
```

### onMutate (optional)

This allows the useConnect Hook to pre-process the parameters given to the connect function before the account is connected, in the event those parameters would result in a connection failure. The return value of this function is used by both the onError and onSettled function.

```ts
import { useConnect } from '@casperdash/useWallet/react';

function App() {
  const { connect } = useConnect({
    onMutate(connector) {
      console.log('Connect');
    },
  });
}
```

### onSuccess (optional)

Function that is invoked when the connection to the account is successful.

```ts
import { useConnect } from '@casperdash/useWallet/react';

function App() {
  const { connect } = useConnect({
    onSuccess() {
      console.log('Connect');
    },
  });
}
```

### onSettled (optional)

Function to invoke when the connection attempt has completed, providing information about success or failure of the connection.

```ts
import { useConnect } from '@casperdash/useWallet/react';

function App() {
  const { connect } = useConnect({
    onSettled(data, error) {
      console.log('Settled', { data, error });
    },
  });
}
```
